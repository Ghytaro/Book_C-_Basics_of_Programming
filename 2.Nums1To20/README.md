# Текущий проект: "Числа от 1 до 20"  

Сегодняшнее упражнение посвящено одной из важнейших конструкций в программировании - циклам. Оно знакомит с базовым способом повторения действий, что критически важно для автоматизации рутинных операций.  

----------

# Задание:  

Написать программу на языке C++, которая выводит на консоль числа от 1 до 20, каждое на отдельной строке.  

----------

# Почему это упражнение важно?

Это, на первый взгляд, простое задание имеет огромное значение для понимания программирования, поскольку оно:
- Вводит понятие цикла: Я осваиваю, как заставить программу повторять действия определённое количество раз без написания одного и того же кода многократно.  
- Знакомит с инкрементом: Учусь использовать операции увеличения числа (например, i++), которые часто применяются в циклах для подсчёта и итерации.  
- Закрепляет вывод данных: Практикую использование std::cout и std::endl для форматированного вывода информации на консоль.  
  
----------

# Моё решение: Использование цикла for

Для выполнения этого задания я использовала цикл for. Он идеально подходит, когда заранее известно, сколько раз нужно повторить действие (в данном случае, 20 раз). Несмотря на то, что книга предлагает использовать 20 строк вывода с std::cout, использование цикла с счётчиком значительно улучшает читаемость и сокращает количество кода.
