# Текущий проект: "Треугольник из 55 звёзд"  

Это задание — шаг в мир вложенных циклов. Если предыдущие задачи помогли мне понять, как работают одиночные циклы для повторяющихся действий, то "Треугольник из 55 звёзд" показывает, как использовать циклы внутри других циклов для создания сложных узоров и структур.  

---

# Задание:  

Написать программу на языке C++, которая выводит на консоль треугольник из 55 звёздочек, расположенных в 10 рядов.  

---

# Почему это упражнение важно?

Это задание не просто рисует узор; оно является ключевым для понимания:
- Вложенных циклов: Это основной фокус задачи. Я учусь, как один цикл (внешний) может управлять строками, а другой (внутренний) — элементами внутри этих строк.
- Управления выводом: Закрепляются навыки форматированного вывода char (символов) и управления переносом строк (\n или std::endl).
- Логического мышления: Требуется продумать, как количество звёздочек в каждой строке зависит от номера этой строки.
- Оптимизации кода: Вместо того чтобы писать 10 отдельных std::cout команд, мы находим элегантное и эффективное решение.

---

# Моё решение: Вложенные циклы for

Я выбрала решение с использованием двух вложенных циклов for. Этот подход позволяет динамически генерировать нужное количество звёздочек в каждой строке и автоматически переходить на новую строку для следующего ряда.
