# Проект: "Площадь прямоугольника"  

Изначально это задание фокусировалось на базовом вводе данных с консоли (std::cin) и выполнении арифметических операций. Однако я решила значительно расширить его функциональность, добавив пользовательское меню и обработку ошибок ввода, что делает программу более надёжной и удобной.  

Задание (расширенная версия):  

Написать программу на языке C++, которая:  
- Предоставляет пользователю меню для выбора действия (например, "вычислить площадь" или "выйти").  
- Считывает с консоли два числа (a и b) — стороны прямоугольника.  
- Вычисляет и выводит на экран площадь прямоугольника.  
- Обрабатывает некорректный ввод данных (например, если пользователь введёт текст вместо числа), предотвращая сбой программы.

# Почему это упражнение важно?  

Это задание является значительным шагом вперёд, поскольку оно охватывает несколько ключевых аспектов программирования:  
- Ввод данных (std::cin): Осваиваю, как получать информацию от пользователя.  
- Базовые арифметические операции: Применяю оператор умножения (*) для вычисления площади.  
- Циклы (while): Использую бесконечный цикл while для создания интерактивного меню, которое повторяется до тех пор, пока пользователь не решит выйти.  
- Условные операторы (if/else if): Применяю их для обработки выбора пользователя в меню.  
- Обработка исключений (try/catch): Это мощный механизм для перехвата и реагирования на ошибки времени выполнения, особенно на некорректный ввод. Это делает программу гораздо более устойчивой к ошибкам пользователя.  
- Манипуляции с потоком ввода (std::cin.fail(), std::cin.clear(), std::cin.ignore()): Учусь, как восстанавливать состояние потока ввода после ошибок, чтобы программа могла продолжить работу.

# Моё решение: Интерактивный калькулятор площади с обработкой ошибок   

Моя программа предлагает пользователю выбор действий. Если выбран расчёт площади, она запрашивает стороны прямоугольника, выполняет расчёт и выводит результат. Самое главное - она устойчива к ошибочному вводу благодаря использованию try-catch блоков и функций для очистки потока ввода.  

# Как это работает?
- Основной цикл while(true): Гарантирует, что меню будет отображаться снова и снова, пока пользователь не выберет опцию выхода.  
- try блок: Код, который может вызвать ошибку (например, некорректный ввод std::cin), помещается сюда.  
- std::cin.fail(): Эта функция проверяет, находится ли поток ввода в "ошибочном" состоянии (например, если вместо числа был введен текст).  
- std::runtime_error(...): Если std::cin.fail() возвращает true, программа выбрасывает исключение, содержащее сообщение об ошибке.  
- catch блок: Перехватывает выброшенное исключение.  
- std::cerr << "Error: " << e.what() << std::endl;: Выводит сообщение об ошибке на стандартный поток ошибок.  
- std::cin.clear();: Сбрасывает все флаги ошибок на std::cin, возвращая его в "рабочее" состояние. Без этого std::cin будет продолжать сообщать об ошибке, и любые последующие попытки ввода будут игнорироваться.  
- std::cin.ignore();: Очищает буфер ввода. Если пользователь ввел "abc\n", "abc" вызывает ошибку, а \n остаётся в буфере. ignore() удаляет все символы из буфера до следующего символа новой строки, предотвращая повторное считывание ошибочного ввода в следующей итерации.
